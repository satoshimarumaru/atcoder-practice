# atcoder-practice

競技プログラミング学習用のリポジトリです。AtCoderの問題を解いた解答をまとめています。

## ディレクトリ構成

```
atcoder-practice/
├── ABC/   # AtCoder Beginner Contest
├── ARC/   # AtCoder Regular Contest
└── AGC/   # AtCoder Grand Contest
```

## 使い方

各コンテストのディレクトリ内に、問題ごとのディレクトリを作成して解答ファイルを配置します。

例: `ABC/ABC001/A.py`

## テンプレート

[template.py](./template.py) に基本的な入力処理のテンプレートを用意しています。
